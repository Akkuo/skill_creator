# 範例：三峰接龍 玩法說明

**輸出：**

## 1. 遊戲概覽 (Game Overview)
- **核心定位**：以撲克牌點數連接為核心、關卡制的單人益智接龍遊戲
- **核心體驗**：在有限的底牌資源下，透過巧妙的接龍路線清空牌桌，獲得乾淨收尾的高成就感

## 2. 玩法概要 (Core Gameplay)

### 2-A. 核心物件與場景元素 (Entities)
- **牌桌區 (Grid)**：由多張撲克牌依疊層結構排列，分為覆蓋狀態（無法操作）與亮牌狀態（可操作）兩種
- **玩家持有牌 (Current Card)**：玩家手上當前持有的一張牌，顯示在玩家操作區，是每次連接的基準點
- **底牌堆 (Deck)**：玩家開場擁有的固定數量抽牌資源，每次消耗一張以更換持有牌，用盡後不補充
- **疊層壓柱關係 (Stack)**：牌桌上的牌具有上下堆疊關係，被壓住的牌為覆蓋狀態，上方的牌被移走後才能翻開

### 2-B. 核心操作循環 (Core Loop)
1. **觀察 (Observe)**：掃描牌桌上所有亮牌的點數，與玩家持有牌做比對，辨識哪些牌的點數與持有牌相鄰（+1 或 -1，A 與 K 視為相鄰）
2. **決策 (Decision)**：選擇連接哪張亮牌——需考量「連接後是否能翻開更多牌」以及「底牌堆是否還需要保留以應急」
3. **執行 (Execute)**：點擊目標亮牌，該牌轉移成新的持有牌，原位置若下方有牌則自動翻開為亮牌狀態
4. **達成 (Goal)**：完成一次連接即獲得即時視覺回饋，連續連接不中斷為最優策略；清空牌桌則觸發通關結算

### 2-C. 局外進度循環 (Secondary Loop / Meta-game)
- **資源產出**：通關後根據剩餘底牌數量與過關時間獲得「金幣」與「星星」。
- **資源消耗**：金幣可用於關卡內購買「額外底牌（+5張）」或「撤銷道具」，以及在商店解鎖不同的「撲克牌背造型」；星星用於解鎖下一章節的地圖。
- **長期目標**：推進世界級別地圖進度，解鎖所有風格迥異的主題場景，並收集全套特殊卡背。

## 3. 遊戲流程 (Game Rules & Constraints)
- **勝負條件**：
  - **Win**：清空牌桌上所有撲克牌
  - **Lose**：底牌堆耗盡，且持有牌無任何亮牌可接龍
- **獎勵機制**：通關後銜接平台結算，提供關卡星級評等與對應獎勵
- **限制條件與防呆**：
  - 無時間限制，純邏輯回合制
  - 僅能點擊亮牌狀態的牌，覆蓋牌無法被選中
  - 點擊底牌堆後強制取代持有牌，無法撤銷
  - 底牌用盡後不可繼續抽牌

## 4. 策略深度與難度設計 (Strategy & Constraints)
- **空間規劃難點**：玩家必須優先清除「深處被壓住的牌」，否則資源消耗後底層牌永遠無法翻開；連接順序直接決定是否能解鎖後續的可操作範圍
- **資源管理壓力**：底牌堆數量有限，每次動用底牌等於放棄當前連接機會；用底牌應急 vs. 靠手牌連接，是核心資源決策
- **時機反應要求**：無即時時間壓力，所有決策為靜態回合制，策略思考時間不受限制
- **難度遞增邏輯**：隨關卡進階，牌桌疊層層數增加（空間規劃難度上升）、初始底牌數量減少（資源壓力上升）、關卡目標牌型更複雜（解法路徑變窄）

## 5. 系統流程圖 (System Flowchart)
- **整體場景動線 (Main Screen Flow)**：
  *(Mermaid 語法，可貼入 Excalidraw 轉換為手繪圖)*
  ```mermaid
  graph TD
    A["啟動遊戲 / Loading"] --> B["主選單"]
    B --> C["關卡選擇畫面"]
    C --> D["遊玩畫面（入場表演 + 開始操作）"]
    D --> E{"牌桌是否清空？"}
    E -- 否 --> F{"底牌耗盡且無法接龍？"}
    F -- 否 --> D
    F -- 是 --> G["失敗結算（跳出接關選項）"]
    G -- 購買底牌/接關 --> D
    G -- 放棄 --> H["失敗結算畫面"]
    H --> B
    E -- 是 --> I["通關結算（星級獎勵）"]
    I --> C
  ```

## 6. Game Juice（爽感回饋設計）

| 觸發操作 | 視覺回饋 (VFX) | 音效回饋 (SFX) | 設計意圖 |
|---------|-------------|-------------|---------|
| 點擊亮牌成功接龍 | 牌張滑移動畫 + 輕微光暈 | 清脆撲克翻牌音效 | 強化每次成功連接的操作手感，讓玩家「上癮於點擊」 |
| 連續接龍不中斷（Combo） | 連擊數字計數器 + 粒子殘影 | 音調逐步升高的連續音效 | 引誘玩家追求更長的連接鏈，延長遊玩時間 |
| 翻開被壓住的新牌 | 牌張翻面動畫 + 短暫高亮邊框 | 厚實的翻牌「啪」聲 | 讓「解鎖隱藏牌」這個行為本身具有獨立的正向回饋 |
| 清空牌桌（通關） | 全畫面彩紙爆炸 + 卡牌飛散演出 | 升調簡短勝利音樂 | 強化「完美清台」的最高成就感，製造分享慾 |
| 底牌耗盡無法接龍（失敗） | 畫面輕微震動 + 牌桌暗化 | 低沉失落短音 | 製造情緒落差，觸發「再來一局」的重試衝動 |

## 7. 擴展與備忘錄 (Roadmap & Notes)
- **未來迭代想法**：每日挑戰模式、全球排行榜（最少底牌通關）、特殊牌型關卡（鬼牌 Joker 機制）
